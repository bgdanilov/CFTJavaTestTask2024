Инструкция к программе.
Добрый день!

1. Основное описано в -help Settings. Использована версия Java 21.
2. В задании были указаны Целые числа, Натуральные числа и Строки в качестве типов сортировки.
Причем, далее файл с натуральными числами предлагалось назвать floats.txt.
Поэтому, я решил, что имелись ввиду вещественные числа (double):
- с нулевой дробной частью - Целые;
- с ненулевой дробной частью - Дробные;
- ну и строки - Строки.
3. Чтобы сделать Jar-файл:
- модуль - FileContentSeparator;
- стартовый класс - ConsoleMain.
